## Descrição do Projeto
- Visualizações: histogramas, boxplots, pairplots e heatmap de correlação


2. **Modelagem**
- Regressão Linear (simples e múltipla – `statsmodels`)
- Regressão Polinomial (`sklearn`)
- Regressão Logística
- Naive Bayes Gaussiano


3. **Avaliação**
- Regressão: MAE, RMSE, R², diagnóstico de resíduos, VIF
- Classificação: Accuracy, Precision, Recall, F1-score, AUC-ROC, Matriz de Confusão


4. **Otimização**
- Validação cruzada
- GridSearch (sklearn)
- Comparação e tuning com PyCaret

## ▶️ Instalação e Execução


### 1️⃣ Clone o repositório
```bash
git clone https://github.com/Henrique-Goncalves-Dev/Modelagem.git
cd Modelagem
```


### 2️⃣ Crie e ative o ambiente virtual
```bash
python -m venv venv
venv\Scripts\activate
```


### 3️⃣ Instale as dependências
```bash
pip install -r requirements.txt
```


### 4️⃣ Execute o projeto
```bash
python projeto_modelagem_.py
```


Ou abra o notebook no Jupyter:
```bash
jupyter notebook
```
